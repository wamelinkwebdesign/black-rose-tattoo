# Black Rose Tattoo — Demo Site

## Project
Demo website for Black Rose Tattoo And Gallery, Amsterdam Centrum.
Built by WAMELINK for outreach purposes.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Key Config
- basePath: /demos/black-rose — DO NOT change
- trailingSlash: true — DO NOT change
- Custom image loader: image-loader.ts (prefixes all images with /demos/black-rose)

## Deploy
- Vercel project: black-rose-tattoo
- Live via wamelink.design/demos/black-rose/

## Commands
- npm run dev
- npm run build
- npx vercel --prod --yes
